#####Usage
---
start a project or file
```
op webstorm ./MyProjects 
```
set alias for ide
```
op alias webstorm ws
op ww ./MyProjects
```
set default ide
```
op default webstorm
op ./MyProject
```
config a new ide
op config <ide> <idePath>
```
op config vscode "Visual Studio Code.app"
```
